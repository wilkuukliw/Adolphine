## Getting Started

To get a local copy up and running on you machine, follow these simple steps.

### Prerequisites

Install [Node.js](https://nodejs.org/en/) 

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/wilkuukliw/Adolphine.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```
3. Run the project

   ```sh
   nodemon app.js
   ```
